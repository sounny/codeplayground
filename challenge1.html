Lesson: Using console.log() and Strings in JavaScript

In JavaScript, the console.log() function is used to print messages to the console. This is helpful when you want to display information or debug your code.

A string is a type of data that represents text. You create a string by wrapping text in quotes: "Hello World!".

You can use console.log() to print a string to the console, like this: 

console.log("Hello World!");

This will print "Hello World!" to the console.
