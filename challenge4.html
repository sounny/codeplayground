Lesson: Getting User Input with prompt()

In JavaScript, you can ask the user to enter information by using the prompt() function. The function displays a dialog box where the user can type input.

For example: 

let name = prompt("What is your name?");

This will show a dialog box asking "What is your name?". The user can type a name, and that value will be stored in the variable name.

You can then use this input in your code. For example: 

console.log("Hello, " + name + "!");

This code will print a personalized message with the user's input. Now, try asking the user for input and printing it to the console!
