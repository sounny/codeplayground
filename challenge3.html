Lesson: Arrays and Loops in JavaScript

In JavaScript, an array is a special variable that can hold multiple values. You can create an array by listing the values inside square brackets [].

For example: 

let fruits = ["apple", "banana", "orange"];

This array contains three fruits: "apple", "banana", and "orange". Each item in the array has an index, starting at 0.

You can loop through an array to access each item using a for loop. For example: 

for (let i = 0; i < fruits.length; i++) {
  console.log(fruits[i]);
}

This code will print each fruit in the array to the console.
